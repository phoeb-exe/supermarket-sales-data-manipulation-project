# 🧹 Data Manipulation with Pandas

## 📌 Overview

This project focuses on **data wrangling (data manipulation)** using Python and Pandas to transform raw data into a structured and analysis-ready format.

In addition to manipulation, the project also includes **basic exploratory analysis (EDA)** and **feature preprocessing**, such as scaling, to prepare data for further analytical or machine learning tasks.

---

## 🎯 Objectives

- Perform **data manipulation** using Pandas  
- Transform raw data into a structured format  
- Apply grouping, aggregation, and reshaping techniques  
- Create new features from existing data  
- Apply basic preprocessing for modeling readiness  
- Extract simple insights from transformed data  

---

## 📂 Dataset

The dataset used in this project contains transactional data (e.g., sales, income, and dates) that is processed and transformed using various data manipulation techniques.

> 📌 All dataset exploration and structure details are available inside the notebook.

---

## 🛠️ Tools & Libraries

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 🔧 What’s Done in This Project

### 🧩 Data Manipulation (Core)

- Filtering data based on conditions  
- Sorting values  
- Grouping and aggregation (`groupby`, `agg`)  
- Merging datasets (`merge`)  
- Concatenating data (`concat`)  
- Reshaping data:
  - Pivot tables  
  - Crosstab  
- Creating new features (e.g., extracting month from date)  

---

### 📊 Basic Exploratory Analysis

- Dataset structure inspection (`info`)  
- Summary statistics  
- Simple metric calculations (e.g., averages, percentages)  

---

### 📈 Visualization (Supporting Analysis)

- Distribution plot (histogram)  
- Trend visualization (line plot)  

---

### ⚙️ Preprocessing

- Feature scaling:
  - Standardization (`StandardScaler`)  
  - Normalization (`MinMaxScaler`)
